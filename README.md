# Progetto Monumenti: Catanzaro 🏛️

Un sito web vetrina dedicato alla scoperta del patrimonio storico e culturale della città di Catanzaro. Il progetto permette di visualizzare una galleria di monumenti accompagnati da descrizioni dettagliate sulla loro storia.

## 🌟 Caratteristiche
* **Sito Vetrina:** Interfaccia pulita dedicata alla valorizzazione dei monumenti locali.
* **Database Dinamico:** Utilizzo di PHP e MySQL per la gestione e il recupero dei contenuti storici e delle immagini.
* **Design Responsive:** Layout adattabile a diversi dispositivi grazie all'integrazione di Bootstrap.

## 🛠️ Tecnologie Utilizzate
* **Frontend:** HTML5, CSS3, [Bootstrap](https://getbootstrap.com/)
* **Backend:** PHP
* **Database:** MySQL
* **Ambiente di Sviluppo:** Sublime Text
